# Research Synthesis

Define the question and decision context first. Track sources, dates,
definitions, and uncertainty. Prefer primary sources when accuracy
matters.

Separate facts, interpretations, and recommendations. Compare options
on the criteria the user actually cares about, including tradeoffs and
unknowns.

When information may be current or time-sensitive, verify it before
answering and cite the sources used.
