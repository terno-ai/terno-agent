# Task Planning

Clarify the goal, constraints, success criteria, and known unknowns.
Split work into independently verifiable steps and identify the critical
path.

Keep plans executable. Each step should have an owner, output, and clear
done condition when the context calls for it.

Revise the plan as evidence changes. Surface risks early, especially
dependencies, irreversible decisions, and assumptions that could change
the direction of the work.
