# Data Cleaning

Profile before transforming. Identify missing values, invalid values,
type inconsistencies, duplicated records, category drift, timezone and
currency issues, and key uniqueness problems.

Make transformations explicit and reversible where possible. Preserve
raw inputs, document assumptions, and avoid deleting records unless the
rule is clear.

After cleaning, verify row counts, key uniqueness, null rates, value
ranges, and aggregate totals against the raw data.
