# Machine Learning

Clarify the prediction target, unit of prediction, label timing, leakage
risk, evaluation metric, baseline, and deployment constraint before
choosing a model.

Split data in a way that matches real use, especially for time series,
users, accounts, or grouped observations. Treat leakage as a first-class
failure mode.

Start with a simple baseline. Compare models with appropriate metrics,
confidence intervals or repeated validation when useful, and error
analysis by segment. Prefer interpretable improvements over complexity
without measurable gain.
